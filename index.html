Desenvolvimento Web:

O desenvolvimento Web é o ramo da informática que estuda o desenvolvimento de soluções na web,
incluindo o design, a usabilidade, funcionalidade, segurança e integração de back-end, desenvolvimento web é o processo de codificação dos sites da internet, ou seja, toda a programação desses portais, com seus respectivos elementos e linguagens.
Outros pontos como o armazenamento de banco de dados, a cibersegurança e a integração com outros sistemas também estão inclusos nessa atividade.

Sistemas Operacionais:

O sistema operacional introduz uma camada de abstração entre o hardware e o usuário, que transforma comandos no mouse, teclado e solicitações do sistema,
como gerenciamento de recursos (CPU, memória RAM), em linguagem de máquina, enviando instruções ao processador.
Terá uma visão geral dos sistemas operacionais, bem como aprenderá conceitos fundamentais e como é feito o gerenciamento de recursos de hardware e software do computador.

Linguagem de Programação:

A linguagem de programação é um sistema de comunicação estruturado, composto por conjuntos de símbolos, palavras-chave, regras semânticas e sintáticas que permitem o entendimento entre um programador e uma máquina.
Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução.
O código de programação serve para criar programas de computador, que podem realizar uma ampla variedade de tarefas.
Desde a criação de um simples programa de cálculo até o desenvolvimento de sistemas complexos, o código é a base para a criação de software.

Manuteção de Computadores:

A manutenção de computadores tem como objetivo monitorar a qualidade das máquinas, analisando seu funcionamento e fazendo reparos. Ela é responsável tanto pela parte externa quanto a interna dos equipamentos, garantindo desde atualização de sistemas até a limpeza das peças.

Segurança de Sistemas:

A segurança de sistemas é um campo que abrange a proteção de dados e informações contra acessos nao autorizados, uso indevido, alterção, divulgação ou detruição. O estudo dessa área envolve a compreensão e aplicação de medidas técnicas, organizacionais e comportamentais para garantir a confidencialidade, integridade e disponibilidade das informações.

Rede de Computadores:

Rede de computadores abrange a compreensão e a aplicação de tecnologias e protocolos para conectar dispositivos eletrônicos, permitindo a comunicação e o compartilhamento de recursos, principais tópicos: fundamentos de redes, topologia de rede, protocolos de comunicação: aprender sobre TCP/IP,HTTP,HTTPS,FTP.
